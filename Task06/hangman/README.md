# Hangman

Веб-игра "Виселица". Угадайте слово, выбирая буквы на экране.

## Запуск

Игра доступна онлайн: [https://Aidar-555.github.io/hangman-web/](https://Aidar-555.github.io/hangman-web/)


## Правила

1. Загадано слово, и ваша цель — угадать его буквы.
2. За каждую неправильную букву рисуется часть виселицы.
3. Игра продолжается, пока вы не угадаете все буквы или человечек не будет нарисован полностью.
4. Можно начать новую игру кнопкой **New Game**.
