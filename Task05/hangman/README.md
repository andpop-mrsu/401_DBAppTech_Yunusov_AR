Hangman
Консольное приложение для игры в "Виселицу". Цель игры — угадать загаданное слово, предлагая буквы по очереди.

Системные требования
PHP 7.4 или выше
Composer 2.0 или выше
Включенное расширение extension=mbstring в файле php.ini
Как установить и запустить
Способ 1. Локальная установка (клонирование репозитория)
Склонируйте репозиторий:

text

git clone https://github.com/aidar555/hangman.git
Откройте директорию проекта:

text

cd hangman
Загрузите зависимости:

text

composer install
Запустите приложение:

text

php bin/hangman
Способ 2. Глобальная установка (через Packagist)
Установите пакет глобально:

text

composer global require aidar555/hangman
Запуск осуществляется командой:

text

hangman
Доступные параметры запуска
Параметр	Назначение
-n, --new	Начать новую игру (используется по умолчанию)
-l, --list	Показать список всех сохранённых партий (в разработке)
-r <id>, --replay <id>	Воспроизвести игру по её идентификатору (в разработке)
-p <имя>, --player <имя>	Задать имя игрока (по умолчанию "Player")
-h, --help	Вывести справочную информацию
Правила игры
Перед стартом можно задать имя игрока, используя параметр -p или --player
Игрок вводит буквы латинского алфавита по одной, отвечая на запросы программы
После каждого хода:

Выводится актуальное состояние слова (неотгаданные буквы скрыты)
Показывается изображение виселицы с учётом количества ошибок
Игра завершается при:

Победе — слово полностью отгадано
Поражении — исчерпан лимит попыток (человечек полностью нарисован)
По окончании партии:

Результат сохраняется в базу данных SQLite: дата, имя игрока, загаданное слово, результат (won/lost)
Каждый ход записывается с номером попытки, введённой буквой и результатом проверки
Дополнительный функционал:

--list — отображение всех сохранённых игр с основными данными
--replay — пошаговое воспроизведение выбранной партии
