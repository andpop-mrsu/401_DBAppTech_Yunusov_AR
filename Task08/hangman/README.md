# Hangman

Веб-игра "Виселица". Угадайте слово, выбирая буквы на экране.

# Запуск

Перейдите в каталог проекта (где находится папка `public`) и запустите сервер:
   ```bash
   cd Task08\hangman
   php -S localhost:3000 -t public
   ```
   Откройте браузер и перейдите по адресу:
   http://localhost:3000/
   


# Правила

1. Загадано слово, и ваша цель — угадать его буквы.
2. За каждую неправильную букву рисуется часть виселицы.
3. Игра продолжается, пока вы не угадаете все буквы или человечек не будет нарисован полностью.
4. Начать новую игру кнопкой **New Game**.

# Режимы игры

- **New Game** — начать новую игру с случайным словом.
- **Saved Games** — просмотр всех сыгранных игр и истории введённых букв.
- **Replay** — просмотр всех попыток игрока в конкретной игре (не позволяет играть заново, только просмотр истории ходов).
- **Info** — отображение справки с описанием возможностей игры.

# Сохранение и статистика

- Информация об игре и история ходов сохраняются на сервере в базе данных SQLite.
- Можно просматривать историю ходов каждой игры через режим **Saved Games**.
